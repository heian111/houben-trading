# Houben Trading Co., Ltd. - Company Website

Bridging Chinese manufacturing excellence with global markets.

## Tech Stack

- Pure HTML5 + CSS3 + Vanilla JavaScript
- Font Awesome 6 for icons
- Google Fonts (Inter)
- Hosted on GitHub Pages (free)

## How to Deploy

### Option 1: GitHub Pages (recommended)
1. Create a GitHub repo named `houben-trading` or `<username>.github.io`
2. Push all files to the `main` branch
3. Go to Settings → Pages → deploy from `main` branch root
4. Your site will be live at `https://<username>.github.io/houben-trading/`

### Option 2: Any web hosting
Just upload the entire folder to any web server or hosting provider.

## Customization

Before going live, replace:
- Email in contact form (formsubmit.co in contact.html line 57)
- Phone number
- Product images (replace gradient placeholders with real photos)
- Any placeholder text

## Contact Form

The contact form uses FormSubmit.co — a free email-based form service.
After deployment, update the form action URL in contact.html to your real email:
`https://formsubmit.co/your-email@gmail.com`
